# MoonFRP Configuration Examples

This directory contains example configuration files showing the actual structure of files generated by MoonFRP.

## Files

### frps.toml
Server configuration file. This is generated when you set up an FRP server.

**Location**: `/etc/frp/frps.toml`

**Key Settings**:
- `bindAddr`: Server bind address (default: 0.0.0.0)
- `bindPort`: Server bind port (default: 7000)
- `auth.token`: Authentication token (shared with clients)
- `webServer`: Dashboard configuration (port 7500)

### frpc-*.toml
Client configuration files for multi-IP setup. Each file is named based on the last octet of the server IP address.

**Examples**:
- `frpc-177.toml` - For server IP `185.177.177.177`
- `frpc-32.toml` - For server IP `185.32.32.32`

**Location**: `/etc/frp/frpc-*.toml`

**Key Settings**:
- `serverAddr`: Server IP address (different for each file)
- `serverPort`: Server port (usually same: 7000)
- `auth.token`: Authentication token (must match server)
- `user`: Client username (e.g., `moonfrp-177`)
- `webServer.port`: Unique port per client (7400 + last octet)

**Proxy Configuration**:
Each file can contain multiple `[[proxies]]` sections for different local services.

## Multi-IP Setup Example

When you configure multiple server IPs:
```bash
Server IPs: 185.177.177.177,185.32.32.32
Server Ports: 7000,7000
Client Ports: 8080,8081
```

The following files will be created:
- `frpc-177.toml` - Connects to `185.177.177.177:7000`
- `frpc-32.toml` - Connects to `185.32.32.32:7000`

Each file has:
- Different `serverAddr` (the IP)
- Same `serverPort` (7000)
- Different `user` (moonfrp-177, moonfrp-32)
- Different `webServer.port` (7577, 7432)
- Different proxy names (tcp_8080-177, tcp_8080-32)

## File Naming Convention

The naming convention uses the **last octet** of the IP address:
- IP `185.177.177.177` → filename `frpc-177.toml`
- IP `185.32.32.32` → filename `frpc-32.toml`
- IP `192.168.1.100` → filename `frpc-100.toml`

This makes it easy to identify which configuration file belongs to which server IP.
